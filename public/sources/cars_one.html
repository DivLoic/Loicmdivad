This school project propose to study a dataset of car crash around Paris. The .csv file with all the records can be download from opendata.paris.fr. Each lines represente a crash, and the columns stand for information like: address, date of the accident type of vehicule and so on...

date ... adresse type
19/03/2012 ... 245, RUE LA FAYETTE VL
02/06/2012 ... 18, RUE DE LA CHAPELLE Moto>125

In order to respond to the project requirement, the file is stored in HDFS and read with Spark. But this could be done with a more simple tool like a Pandas dataframe (Python). Finally you can restitute the result of the processing with a cool data visualisation.
